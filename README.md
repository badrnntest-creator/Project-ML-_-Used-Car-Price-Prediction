# Prédiction du Prix de Vente d'une Voiture d'Occasion

Projet réalisé dans le cadre du **Module 5 : Machine Learning — EHTP, MSDE Édition 7**.

Ce projet a permis de mettre en oeuvre un pipeline Machine Learning complet applique a la prediction du prix de vente de voitures d’occasion. Partant d’un dataset de 370 000 annonces issues de la plateforme Kaggle (eBay Kleinanzeigen), nous avons conduit une analyse exploratoire rigoureuse, un preprocessing adapte, entraine 11 algorithmes de regression et optimise les 3 meilleurs par RandomizedSearchCV
